Introducing the Modulous Mod Manager

Features

Easy sharing

Installing a mod is as easy as clicking a link, clicking a Modulous Web Protocol link will open your manager and guide you through installation.

Easy to use

The mod manager takes out the tedious part of installing mods.

Auto-updating

The Mod Manager can update itself, so you are always on the latest version.

BrawlEX compatible

Installing a new character/stage in BrawlEX is not an easy task, but we make it easy requiring no interaction to install any BrawlEX based mod.
