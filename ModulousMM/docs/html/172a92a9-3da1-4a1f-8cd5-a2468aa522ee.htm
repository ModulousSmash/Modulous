How to make a mod work with the mod manager

Making mods work with the mod manager is quite simple, here's a guide on how to do it.

Preparing your working folder

  1. You must create a mod root and an empty file ending in .lua, the mod root is basically the root of the generated zip file, and the root for all your installation code to be relative to.

  2. Now you must write a installation script inside the .lua file you created, you can look at documentation on how to write one here.

  3. You also must copy your mod's files to your mod root folder, remember all paths are relative, so if you have a file named link.txt then in code that file would be link.txt.

Packaging your mod

  1. Go to the modulous mod manager settings and click "Open Mod Creation Tools", inside there fill all your information, select your Lua script and your mod root.

  2. Now after clicking save you will have a zip file, to test it, create a folder and put brawlex inside just like you would do in a SD Card, go back to settings, change your SD Card location to this new folder and then click on Load Debugging Console.

  3. To test your mod, simply click the second option in the main mod manager window and select your ZIP file, if it all install, it's ready to upload to modulous.
