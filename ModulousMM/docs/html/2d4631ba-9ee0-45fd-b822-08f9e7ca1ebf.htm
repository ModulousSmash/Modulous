How to write an installation script

For your mod to install you need to make a Lua script that will tell the manager what to do during installation, here is an example

Copy
SDCard.copy_from_mod_to_sd_card("File!.txt", "")
SDCard.copy_from_mod_to_sd_card("im a folder/", "/")
CharacterManager.add_character_from_mod_root("ft_oldlink.rel", "Cosmetic40.dat", "Slot40.dat", "Fighter40.dat", "CSSSlot40.dat", "Link" , "Oldlink")

This code copies a file named "File!" and a folder named "im a folder" into the sd card root, it also adds a character named Oldlink and it's respective CSS files from the mod root.
